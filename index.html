Pteridophyta Token (PT)

基于币安智能链（BSC）的创新型 ERC20 代币，具备自动解锁和用户激励机制。

项目概述

Pteridophyta Token (PT) 是一个运行在币安智能链（BSC）上的 ERC20 代币，具有以下特点：

  • 自动解锁：基于智能合约，无需中心化服务或人工干预。
  • 用户激励：转账触发者获得解锁代币的 0.15% 作为奖励。
  • 低维护成本：开发者无需为解锁支付额外 Gas 费用。
  • 安全性：基于 OpenZeppelin 4.9.0，开发者地址不可更改。

注意：代币采用动态供应模型，持续增发可能导致通胀风险。

核心机制

机制值说明
解锁触发条件非铸造/非销毁转账通过 transfer 或 transferFrom 触发
解锁间隔3600 区块约 3 小时（基于 BSC 3 秒/区块）
解锁量100 PT基础解锁单位
用户奖励0.15%转账触发者获得解锁量的 0.15%
开发者分配99.85%剩余代币分配给开发者地址
初始供应量5000 PT部署时铸造给开发者地址
事件记录TokensUnlocked记录触发者、区块号、解锁量和奖励

用户交互流程

graph TD A[用户发起转账] --> B{检查解锁条件} B -->|满足解锁间隔| C[计算错过的周期数] C --> D[计算总解锁量] D --> E[用户奖励 = 解锁量 × 0.15%] E --> F[铸造奖励给触发者] F --> G[铸造剩余代币给开发者] G --> H[更新解锁状态] H --> J[触发 TokensUnlocked 事件] J --> I[完成原始转账] B -->|未满足间隔| I

注意：用户可通过零值转账触发解锁。

经济模型分析

用户收益

  • 奖励：每次解锁 0.15 PT（100 PT × 0.15%）。
  • Gas 成本：触发解锁的转账 Gas 费用较高（因包含两次铸造，不过绝对是有赚的）。

安全性

基于 OpenZeppelin 4.9.0 标准库，开发者地址不可更改，构造函数验证非零地址。

网络依赖

基于 BSC 3 秒/区块（3600 区块 ≈ 3 小时）。若部署到以太坊（12 秒/区块），解锁时间约为 12 小时。
