Pteridophyta Token (PT)

An ERC20 token on Binance Smart Chain with automated unlocking and user incentives.

Project Overview

Pteridophyta Token (PT) is an ERC20 token on Binance Smart Chain (BSC) featuring:

  • Automated Unlocking: Smart contract-based, no centralized services.
  • User Incentives: 0.15% of unlocked tokens rewarded to transfer initiators.
  • Low Maintenance: Developers incur no Gas fees for unlocking.
  • Security: Built with OpenZeppelin 4.9.0, immutable developer address.

Note: Continuous minting may lead to inflation risks.

Economic Model

User Benefits

  • Reward: 0.15 PT per unlock (100 PT × 0.15%).
  • Gas Cost: Higher for unlock transfers due to minting.
  • Breakeven: PT price > $0.40 (assuming ~$0.06 Gas cost).

Developer Benefits

  • Passive Income: 99.85 PT per unlock, no Gas cost.
  • Risk: No supply cap, potential value dilution.

Security

Built with OpenZeppelin 4.9.0, immutable developer address, non-zero address validation.

Network Dependency

Assumes BSC 3s/block (3600 blocks ≈ 3 hours). On Ethereum (12s/block), unlocks take ~12 hours.
